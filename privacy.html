개인정보처리방침

최종 수정일: 2026년 2월 27일

YouTube API Services 고지: Fitz Intelligence는 YouTube API Services를 사용하여 동영상 정보를 제공합니다. 본 서비스를 이용함으로써 귀하는 YouTube 서비스 약관 및 Google 개인정보처리방침에 동의하는 것으로 간주됩니다.

1. 수집하는 정보

Fitz Intelligence는 다음과 같은 정보를 수집합니다:

1.1 직접 수집 정보

  • 회원가입 정보: 이메일 주소
  • 서비스 이용 정보: 등록한 키워드, 관심 주제

1.2 YouTube API Services를 통해 수집되는 정보

  • YouTube 동영상 제목, 설명, 채널명
  • 조회수, 업로드 날짜, 구독자 수
  • 동영상 썸네일 이미지

중요: Fitz Intelligence는 YouTube 동영상 정보를 브리핑 목적으로만 사용하며, 사용자의 YouTube 계정 정보, 시청 기록, 또는 개인 데이터에 접근하지 않습니다.

1.3 자동 수집 정보

  • 접속 로그 (IP 주소, 접속 시간, 브라우저 정보)
  • 서비스 이용 기록 (브리핑 조회 이력, 피드백)

2. 정보의 이용 목적

수집된 정보는 다음 목적으로 사용됩니다:

  • 맞춤형 뉴스 브리핑 제공
  • YouTube 동영상 큐레이션 및 추천
  • 서비스 개선 및 품질 향상
  • 사용자 피드백 분석 및 AI 에이전트 최적화
  • 고객 지원 및 문의 응대
  • 서비스 공지사항 전달

3. 정보의 공유 및 제공

3.1 외부 제공

Fitz Intelligence는 사용자의 개인정보를 제3자에게 판매하거나 제공하지 않습니다. 단, 다음의 경우 예외로 합니다:

  • 사용자의 명시적 동의가 있는 경우
  • 법령에 의해 요구되는 경우

3.2 제휴 서비스

본 서비스는 다음 제3자 서비스를 이용합니다:

  • YouTube API Services: 동영상 정보 제공 (Google 개인정보처리방침 참조)
  • Supabase: 데이터베이스 및 인증 서비스
  • Google Gemini AI: 콘텐츠 분석 및 요약

3.3 YouTube Data API 정책 준수

본 서비스는 YouTube API Services를 통해 제공되는 데이터를 다음과 같이 처리합니다:

  • YouTube 데이터는 브리핑 생성 및 표시 목적으로만 사용됩니다
  • 사용자가 YouTube 콘텐츠와 상호작용할 때 YouTube 플레이어 또는 YouTube 웹사이트로 연결됩니다
  • YouTube 데이터는 30일 이상 저장되지 않으며, 캐시 데이터는 정기적으로 삭제됩니다

4. 정보의 보관 및 보호

4.1 보관 기간

  • 회원 정보: 회원 탈퇴 시까지
  • 브리핑 데이터: 최대 1년
  • YouTube API 데이터: 최대 30일 (캐시)
  • 접속 로그: 3개월

4.2 보안 조치

  • 암호화된 통신 (HTTPS)
  • 데이터베이스 암호화
  • 접근 권한 관리
  • 정기적인 보안 점검

5. 사용자 권리

사용자는 다음 권리를 행사할 수 있습니다:

  • 열람 요구권: 본인의 개인정보 확인
  • 정정 요구권: 잘못된 정보 수정 요청
  • 삭제 요구권: 개인정보 삭제 요청
  • 처리 정지 요구권: 개인정보 처리 중지 요청

권리 행사를 원하시면 아래 연락처로 문의해주세요.

6. Google 사용자 데이터 정책 준수

Fitz Intelligence는 Google API Services User Data Policy를 준수하며, 다음 사항을 엄격히 이행합니다:

  • 제한적 사용: YouTube 데이터는 명시된 기능 제공에만 사용됩니다
  • 투명성: 사용자에게 데이터 사용 목적을 명확히 고지합니다
  • 보안: 적절한 보안 조치를 통해 데이터를 보호합니다

7. 쿠키 및 추적 기술

본 서비스는 다음과 같은 기술을 사용합니다:

  • 필수 쿠키: 로그인 상태 유지
  • 분석 쿠키: 서비스 개선을 위한 이용 통계 (익명화)

8. 아동의 개인정보 보호

Fitz Intelligence는 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 만 14세 미만 아동이 개인정보를 제공한 사실을 발견한 경우 즉시 삭제합니다.

9. 개인정보처리방침 변경

본 방침이 변경될 경우, 서비스 내 공지사항 또는 이메일을 통해 사전 고지합니다. 중요한 변경 사항은 최소 7일 전 공지하며, 변경된 방침은 공지한 날짜로부터 효력이 발생합니다.

10. YouTube Data 삭제 요청

사용자는 언제든지 Fitz Intelligence가 보유한 YouTube 관련 데이터의 삭제를 요청할 수 있습니다. Google의 보안 설정 페이지(https://security.google.com/settings/security/permissions)를 통해서도 제3자 앱 접근 권한을 철회할 수 있습니다.
